### 【第三部分：浏览商品 - 分页功能】（2-3分钟）★重点

#### 动作
- 点击"浏览商品"按钮

#### 讲解词
```
接下来演示浏览商品功能。

（点击"浏览商品"按钮）

这是浏览商品界面。

【展示数据规模】
（指向屏幕上方）
大家可以看到，系统中目前有141件在售商品。

由于商品数量较多，如果一次性全部显示，
界面会非常拥挤，用户体验不好。

【分页功能介绍】
因此，我实现了分页浏览功能。

（指向底部分页信息）
这里显示"第1/18页（共141件在售商品）"。

每页显示8件商品。
为什么是8件呢？
这是我综合考虑了界面高度和用户体验后确定的。
窗口高度600像素，列表区域约320像素，
每行40像素，正好可以显示8行。

【演示翻页】
（点击"下一页"按钮）

点击"下一页"，页面跳转到第2页。

（指向分页信息）
可以看到，页码信息更新为"第2/18页"，
显示的是第9-16件商品。

（再点击一次"下一页"）

继续点击，到第3页。
页码变为"第3/18页"。

（点击"上一页"按钮）

点击"上一页"可以返回前一页。
回到第2页。

【技术实现讲解】
这个分页功能的实现原理是：

1. 使用一个滚动偏移量变量 scrollOffset 来记录当前显示的起始位置
   - 第1页：scrollOffset = 0，显示第0-7个商品
   - 第2页：scrollOffset = 8，显示第8-15个商品
   - 第3页：scrollOffset = 16，显示第16-23个商品

2. 每次翻页时：
   - 点击"下一页"：scrollOffset += 8
   - 点击"上一页"：scrollOffset -= 8

3. 然后只显示 scrollOffset 到 scrollOffset+7 范围内的商品

4. 同时做边界检查，防止越界：
   - 确保 scrollOffset >= 0
   - 确保 scrollOffset <= 总数量 - 8

【优势说明】
这样做的好处是：

1. 用户体验好
   一次跳转8个商品，不需要多次点击。
   原来如果逐行滚动，查看第3页需要点击16次，
   现在只需要点击2次，效率提升了8倍。

2. 界面清爽
   不会显示过多信息，避免信息过载。

3. 性能好
   只渲染当前页的8个商品，响应速度快。

4. 统一性好
   这个分页功能在三个界面中都有应用：
   浏览商品、搜索结果、交易历史，
   保持了界面交互的一致性。

【进入商品详情】
（点击某个商品行，比如第一行）

点击任意商品，可以进入商品详情页面。
```

#### 屏幕显示

- 浏览商品界面（第2页）
- 浏览商品界面（第3页）
- 浏览商品界面（第2页）
- 商品详情界面

#### 技术点
- 分页算法实现
- 滚动偏移量管理
- 双重索引机制
- 边界检查
- 用户体验优化

#### 注意事项
- 这是重点功能，要讲清楚
- 指向分页信息时要明确
- 讲解原理时可以配合手势
- 强调效率提升（8倍）

原来使用 isprint() 函数判断字符会导致中文输入崩溃，

if (strstr(item.name, keyword) != NULL ||

5. 评价统计：
   - 总评价数：94条
   - 好评数：35条（4-5星）

   - 差评数：36条（1-2星）

【六级信誉等级体系】
系统设计了六级信誉等级：

1. 新手（0-50分）：新注册用户
2. 青铜（51-100分）：有一定交易经验
3. 白银（101-200分）：活跃用户
4. 黄金（201-500分）：信誉良好用户
5. 铂金（501-1000分）：高信誉用户
6. 钻石（1001+分）：顶级信誉用户

这个等级划分参考了淘宝、闲鱼等成熟平台，
等级清晰，用户容易理解。

【信誉分数计算规则】
信誉分数的计算综合考虑两个因素：

1. 交易行为
   - 完成交易：每笔加10分
     鼓励用户诚信交易
   
   - 取消交易：每笔减5分
     惩罚不负责任的行为

2. 用户评价
   根据平均评分调整，范围从-50到+50分：
   - 平均评分 ≥ 4.5：加50分
   - 平均评分 ≥ 4.0：加30分
   - 平均评分 ≥ 3.5：加10分
   - 平均评分 ≥ 3.0：加0分
   - 平均评分 ≥ 2.0：减20分
   - 平均评分 < 2.0：减50分

这样既考虑了交易数量，也考虑了交易质量。

【设计理念】
这个信用评价系统的设计理念是：

1. 建立信任机制
   让每个用户都有信誉档案，
   交易记录和评价都有据可查。

2. 鼓励诚信交易
   完成交易加分，取消交易扣分，
   引导用户负责任地交易。

3. 反映真实信誉
   通过科学的计算规则，
   真实反映用户的信誉状况。

4. 提供决策依据
   其他用户可以根据信誉等级和评价，
   决定是否与该用户交易。
